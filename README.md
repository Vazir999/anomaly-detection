# AI-Based Anomaly Detection for Satellite Operations

Developed during internship at Azercosmos — Azerbaijan's national space company.

## What it does
Detects anomalies in satellite telemetry data using machine learning,
improving the accuracy and efficiency of satellite operations.

## Technologies
- Python
- NumPy, Pandas
- Isolation Forest, SVM (scikit-learn)
- Matplotlib
